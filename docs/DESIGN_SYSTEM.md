# Grupli Design System

## Concepto visual

Grupli debe sentirse como una app real, clara y cercana:

- Fondo cálido casi blanco.
- Texto principal azul navy suave.
- Verde/teal apagado para acciones importantes.
- Tonos sage, lavanda y ámbar muy suaves para estados y categorías.
- Nada de colores chillones.
- Nada de landing dentro de la app.
- Todo debe entenderse en 2 segundos.

## Reglas de interfaz

1. Pantallas centradas con ancho móvil máximo.
2. Cards blancas con borde fino y sombra mínima.
3. Botones grandes, claros y con una sola acción principal por pantalla.
4. Los datos importantes deben ir en bloques visuales: métricas, chips, líneas de datos y cards.
5. Los formularios deben usar etiquetas visibles, placeholders claros y controles fáciles.
6. No duplicar estilos dentro de pantallas: usar `theme/` y `ui/`.

## Paleta

- Navy: títulos y jerarquía.
- Teal: acción principal.
- Sage/Mint: fondos suaves y estados positivos.
- Lavender: detalles secundarios.
- Coral: peligro o errores.
- Amber: avisos.

## Objetivo

La app no debe parecer una web adaptada. Debe parecer una app móvil de organización que puedes usar con una mano.
