# V15.9 — Avisos arriba + navegación limpia + tarjeta cristal

Cambios:

- Quitado el bloque grande de respuestas pendientes del Inicio.
- Añadido botón de campana arriba, junto al menú de tres puntos.
- La campana muestra contador de respuestas pendientes y notificaciones sin leer.
- Al tocar la campana se abre una hoja inferior con:
  - respuestas pendientes
  - notificaciones reales del grupo
- Al tocar una respuesta pendiente se abre el evento.
- Al tocar una notificación se marca como leída y abre el grupo relacionado.
- Eliminada la pestaña Más del navegador inferior.
- El menú de tres puntos superior mantiene el centro de control del grupo.
- Tarjeta de próxima quedada más compacta, con efecto cristal y botones de asistencia más pequeños.
- Sin SQL nuevo.

Inspiración UI:

- Badge/contador tipo Material 3 para avisos.
- Top actions para acciones de pantalla.
- Navegación inferior más limpia: solo secciones principales de uso diario.
- Card con blur suave tipo glassmorphism.
