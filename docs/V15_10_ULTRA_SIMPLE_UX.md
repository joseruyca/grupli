# V15.10 — UX ultra simple + interfaz para todos

Objetivo: hacer Grupli más fácil para usuarios poco cómodos con el móvil.

Cambios principales:
- Menos paneles grandes.
- Menos texto explicativo.
- Menos acciones duplicadas.
- Inicio más directo: próximo plan, resumen y actividad mínima.
- Actividad reciente limitada a 3 elementos.
- Rutinas agrupadas para no repetir el mismo partido muchas veces.
- Tarjeta principal de evento más compacta.
- Botones Voy / Duda / No más ligeros.
- Calendario más compacto: fuera cabecera de métricas y leyenda permanente.
- Agenda del día se mantiene como lugar principal para responder a eventos futuros.
- Finanzas con hero más pequeño y lenguaje más humano.
- Torneos con menos acciones duplicadas.
- Mantiene la funcionalidad actual.

Regla de diseño:
Si algo no ayuda a decidir o actuar en 2 segundos, se reduce o se oculta.
