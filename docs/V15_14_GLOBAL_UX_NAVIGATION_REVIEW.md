# v15.14 — Repaso global de UX y navegación

Objetivo: dejar la app más coherente y fácil de entender antes de seguir añadiendo funciones grandes.

## Cambios incluidos

- Navegación de grupo revisada: la barra inferior incluye ahora Inicio, Agenda, Finanzas, Torneos y Más.
- Textos de pantallas revisados para ser más claros y menos técnicos.
- Mis grupos simplificado: se elimina duplicación de acciones cuando ya hay grupos.
- Pantalla Más más ordenada: invitaciones, miembros, permisos y ajustes quedan como centro de control.
- Agenda renombrada y textos orientados a acción: tocar día, ver planes y crear plan.
- Finanzas más limpia: se quitan tarjetas explicativas repetidas debajo de Gastos y Saldos.
- Estados vacíos rediseñados para ser más legibles y consistentes.
- Header unificado para mejorar jerarquía visual.
- Limpieza menor de warnings por elementos no usados.

## Sin SQL nuevo

Esta fase no requiere cambios de base de datos.
