# v15.25.1 — Limpieza visual real + ajuste de imágenes

Cambios incluidos:

- Se eliminan los fondos con patrón/estampado de las tarjetas principales.
- Se evita usar la foto como fondo teñido con filtro azul encima.
- Las tarjetas de grupo pasan a usar fondo mate sólido y una miniatura limpia de portada.
- El banner del grupo usa fondo sólido y muestra la portada como imagen limpia, sin overlay azul.
- Al elegir portada de grupo se abre un editor de encuadre.
- Al cambiar portada desde ajustes del grupo también se abre el editor de encuadre.
- Al cambiar la foto de perfil se abre un editor de encuadre cuadrado/circular.
- El editor permite ajustar zoom, posición horizontal y posición vertical antes de subir la imagen.
- La portada se exporta en formato banner 16:7.
- El avatar se exporta en formato 1:1.

No requiere SQL nuevo.
