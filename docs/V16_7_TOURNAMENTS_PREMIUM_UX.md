# Grupli v16.7 — Torneos Premium UX

## Objetivo

Hacer Torneos más fácil por fuera y más potente por dentro.

## Cambios

- Modo rápido con plantillas.
- Modo avanzado escondido.
- Plantillas rápidas:
  - Liga sencilla.
  - Liga de pádel.
  - Americano de pádel.
  - Copa rápida.
  - Manual.
- Nueva pantalla de creación con jerarquía más clara.
- Tarjeta de modo organizador en el resumen del torneo.
- Check-in rápido de participantes.
- Pantalla "Panel de edición" para ver la ronda/partidos actuales.
- Textos más humanos y menos técnicos.

## Filosofía

El usuario normal crea un torneo sin pensar. El usuario avanzado puede activar opciones más profundas.
