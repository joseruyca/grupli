# V14.6 — Torneos guiados definitivo

Esta fase convierte la sección Torneos en una experiencia guiada, no en una lista fría.

## Cambios principales

- Hub de torneos más premium:
  - activos
  - finalizados
  - partidos jugados/totales
  - siguiente competición a revisar
- Crear rápido por formato:
  - Liga
  - Copa / Eliminatoria
  - Americano
- Flujo recomendado visible:
  1. Crear
  2. Participantes
  3. Partidos
  4. Resultados
- Estado vacío útil: explica que Grupli evita Excel y guía el proceso.
- Nuevo asistente de creación por pasos:
  1. Nombre
  2. Formato
  3. Tipo de participantes
  4. Revisión
- Preview antes de crear.
- El detalle de torneo mantiene:
  - siguiente paso
  - añadir participantes
  - usar miembros del grupo
  - generar partidos
  - registrar resultados
  - clasificación automática
  - finalizar / reabrir / eliminar

## SQL

No hay SQL obligatorio en esta versión.
