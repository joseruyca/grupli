# Grupli v16.12.7 — Agenda color rebalance

## Objetivo

La Agenda se veía demasiado amarilla cuando había muchos partidos de liga/torneo. Esta versión mantiene los torneos diferenciados, pero sin saturar la pantalla.

## Cambios

- El dorado deja de usarse como fondo grande.
- Los días con torneos usan estructura teal/neutral.
- El dorado queda solo para:
  - icono pequeño de trofeo;
  - chip pequeño "Torneo";
  - puntos del calendario.
- El selector Semana/Mes deja de tener fondo amarillo.
- Los grupos de varios partidos del mismo día ahora son tarjetas blancas/neutras.
- Las filas internas de partidos usan fondo gris/surface, no amarillo.
- Los eventos individuales de torneo ya no tienen tarjeta amarilla ni barra superior dorada.

## Resultado esperado

La Agenda se lee primero como calendario limpio. Los torneos siguen destacando, pero como información especial, no como bloque visual dominante.
