# v6.7 — Detalle de grupo + Miembros

Objetivo: acercar estas pantallas al concepto visual aprobado.

## Detalle de grupo

- Hero card con icono, nombre, rol, tipo y privacidad.
- Código de invitación más visible.
- Botones separados para copiar y compartir.
- Resumen rápido con próxima quedada, miembros, privacidad y mínimo.
- Grid de accesos rápidos: Calendario, Finanzas, Torneos, Miembros.
- Acción de editar grupo solo para admin/owner.
- Acciones destructivas al final.

## Miembros

- Métricas superiores: total, admins y miembros.
- Búsqueda más clara.
- Filtro por todos/admins/miembros.
- Encabezado visual de columnas.
- Cards de miembro más limpias.
- Menú de acciones para admin: hacer/quitar admin, expulsar.
- Leyenda final para entender roles.

## Criterio visual

Mantener el concepto Grupli:
fondo cálido, navy, teal apagado, detalles suaves, cards blancas, bordes finos y jerarquía clara.
