# Grupli v14 — Premium product polish pass

Esta fase no añade una función grande nueva. Mejora la sensación de app premium y la coherencia visual.

## Cambios aplicados

- Sistema visual más consistente:
  - colores base refinados
  - tarjetas más suaves
  - sombras más ligeras
  - inputs más compactos
  - navegación inferior con sombra premium
- Home / Mis grupos:
  - tarjetas de grupos más claras
  - chips de estado más útiles
  - jerarquía visual más cercana a app real
- Inicio de grupo:
  - hero del grupo más compacto
  - menos altura desperdiciada
  - foco en próxima quedada, asistencia y estado del grupo
- Navegación:
  - se mantiene la estructura estable de v12/v13
  - no vuelve AppScreen ni wrappers antiguos
- SQL:
  - no hay migración obligatoria
  - se actualizan checks de seguridad para diagnóstico

## Regla de producto

Grupli debe sentirse como una app móvil real:
- blanca
- compacta
- clara
- sin ejemplos falsos
- con acciones visibles
- con Inicio del grupo como resumen útil
- con Más como zona de utilidades/admin
