# Grupli v12.5 — Calendario + eventos unidos

Esta versión corrige el flujo de quedadas/eventos para que el grupo se organice desde dos sitios coherentes:

## Inicio del grupo
- Muestra el siguiente evento importante.
- Permite responder asistencia directamente: Voy, Duda, No.
- Enseña si el mínimo de asistentes está alcanzado.
- Muestra próximos eventos reales, sin actividad falsa ni ejemplos inventados.

## Calendario
- Es la vista de agenda del grupo.
- Permite cambiar de mes.
- Marca los días que tienen eventos.
- Al pulsar un día, muestra la agenda de ese día.
- Permite crear un evento desde el día seleccionado.
- Cada evento del día permite responder asistencia sin salir del calendario.
- Cada evento puede abrirse para ver detalle completo.

## Crear evento
- Flujo más guiado.
- Plantillas rápidas: Partido, Entrenamiento, Cena del grupo, Reunión.
- Título, fecha, hora, lugar, mínimo de asistentes y notas.
- El mínimo sirve para mostrar si el evento es viable.

## Detalle de evento
- Vista completa del evento.
- Respuesta propia clara: Voy, Duda, No voy.
- Estado de mínimo alcanzado.
- Resumen de asistencia.
- Lista de miembros con estado de respuesta.
- Edición del evento.
- Cancelación del evento.

## Regla de producto
No existe una pestaña principal llamada Eventos. Los eventos viven en:
- Inicio: resumen y próximas decisiones.
- Calendario: agenda completa y creación por día.
