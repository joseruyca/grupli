# Grupli v16.12.8 — Inicio y torneos más limpios

## Cambios

- Eliminada del inicio del grupo la fila de métricas rápidas:
  - Agenda
  - Balance
  - Torneos

Esa sección podía mostrar datos confusos, especialmente balances ya saldados o contadores muy altos de eventos generados por torneos.

- La portada del grupo queda más directa:
  - Próximo plan
  - Actividad reciente

- Eliminada de la pantalla principal de Torneos la sección “Próximos partidos”.

Los partidos se consultan dentro de cada torneo/liga para no mezclar partidos de competiciones distintas en una misma lista.

## SQL

No requiere SQL nuevo.
