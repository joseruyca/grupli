# Grupli v15.25 — Rediseño visual premium basado en mockup

Esta versión aplica la dirección visual aprobada en el mockup:

- tarjetas de grupo oscuras, mate y más sólidas;
- portada/imagen integrada de forma más limpia;
- cabecera del grupo con tarjeta hero más premium;
- tarjeta de próximo plan más destacada, con fondo oscuro y botones de asistencia con colores claros;
- barra inferior con colores semánticos por sección:
  - Inicio: azul navy;
  - Agenda: amarillo/ámbar;
  - Finanzas: verde;
  - Torneos: rojo/coral;
  - Más: morado;
- Finanzas con hero verde, más alineada con la idea de dinero/balance;
- tarjetas generales con sombras, radios y espaciados más modernos.

No añade SQL nuevo. No cambia la lógica de eventos, finanzas, invitaciones ni OSM. Es una capa de interfaz sobre v15.24.1.

## Revisión recomendada

1. Abrir Mis grupos.
2. Entrar a un grupo.
3. Revisar la tarjeta de Próximo plan.
4. Cambiar entre pestañas inferiores.
5. Revisar Finanzas en Gastos/Saldos/Liquidar.
6. Probar en APK para comprobar contraste y tamaño real.
