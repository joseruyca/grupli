# Guía para vídeos de intro y demo de Grupli

Mi recomendación: no metas vídeos pesados dentro de la app. Para la app usa onboarding corto con pantallas estáticas/animadas. Para enseñar y vender la app usa vídeos externos: Play Store, TikTok, Instagram, web y testers.

## Vídeo 1 — Demo corta para testers

Formato: vertical 9:16.
Duración: 20–30 segundos.
Objetivo: que alguien entienda Grupli rápido.

Guion:

1. 0–3s: “Organiza tu grupo en segundos”. Mostrar Mis grupos.
2. 3–8s: “Crea planes y confirma asistencia”. Mostrar Inicio + botones Voy/Duda/No.
3. 8–14s: “Controla gastos como Tricount”. Mostrar Finanzas.
4. 14–20s: “Monta torneos y ligas”. Mostrar Torneos.
5. 20–25s: “Invita con código o enlace”. Mostrar Más.
6. 25–30s: “Grupli: planes, gastos y torneos en un solo lugar”.

## Vídeo 2 — Intro visual para landing/app store

Formato: vertical 9:16.
Duración: 12–18 segundos.
Estilo: premium, limpio, fondo claro, mockups del móvil flotando.

Texto en pantalla:

- “Tu grupo, organizado.”
- “Planes claros.”
- “Gastos sin líos.”
- “Torneos automáticos.”
- “Invita, confirma y listo.”

## Cómo hacerlo fácil

Opción rápida:

1. Graba la pantalla del móvil con la APK.
2. Usa CapCut.
3. Corta silencios y esperas.
4. Añade zooms suaves.
5. Añade textos grandes.
6. Exporta en 1080x1920.

Opción más premium:

1. Graba clips cortos de cada pantalla.
2. Mételos en un mockup de móvil en Canva/CapCut.
3. Añade fondo claro con sombras suaves.
4. Usa música limpia y poco invasiva.
5. Exporta versiones:
   - 9:16 para TikTok/Reels/Shorts.
   - 16:9 para web.
   - 1:1 para post.

## Qué no haría

- Vídeo largo dentro del onboarding.
- Música fuerte.
- Explicar demasiadas funciones.
- Texto pequeño.
- Pantallas con datos reales privados.

## Prompt para generar un vídeo estilo anuncio

“Create a modern vertical app promo video, 15 seconds, clean premium mobile app style, soft white background, dark teal and violet accents, smooth phone mockup transitions, Spanish text overlays: ‘Organiza tu grupo’, ‘Confirma planes’, ‘Divide gastos’, ‘Crea torneos’, ‘Invita y listo’, friendly, elegant, 2026 mobile UI design, no people, no voiceover.”

## Checklist antes de grabar

- Crear grupo demo limpio.
- Usar nombres ficticios.
- Usar fotos de prueba libres o avatares.
- Tener 2–3 eventos.
- Tener 2 gastos.
- Tener 1 torneo con partidos.
- Poner el móvil en modo no molestar.
- Ocultar notificaciones personales.
