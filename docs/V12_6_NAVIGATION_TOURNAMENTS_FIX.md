# Grupli v12.6 — Navegación de grupo y torneos mejorados

## Cambios de navegación

- Se elimina la frase inferior `Estás dentro de [grupo] 🔒`.
- La barra inferior del grupo queda limpia:
  - Inicio
  - Calendario
  - Finanzas
  - Torneos
  - Más
- Las pestañas internas del grupo ya no muestran flecha de volver arriba.
- Si el usuario está dentro de Calendario, Finanzas, Torneos o Más y usa el botón atrás del navegador/móvil, vuelve a **Inicio del grupo**, no a la pantalla de crear/unirse a grupo.
- Solo la pantalla Inicio del grupo mantiene flecha para salir a Mis grupos.

## Cambios en Ligas y Torneos

- El flujo de competición queda más fácil:
  1. Crear competición.
  2. Elegir formato: Liga, Eliminatoria/Copa o Americano.
  3. Elegir participantes: individual, parejas o equipos.
  4. Añadir participantes manualmente o añadir miembros del grupo.
  5. Generar partidos.
  6. Registrar resultados.
  7. Ver clasificación.
  8. Finalizar o reabrir.

## Mejoras nuevas

- Botón para añadir automáticamente los miembros del grupo como participantes.
- Prevención de duplicados al importar miembros.
- Regenerar partidos desde el menú de competición, con confirmación.
- En eliminatorias ahora se exige un cuadro limpio: 2, 4, 8 o 16 participantes.
- Si no hay participantes suficientes, la app indica claramente el siguiente paso.
- La pantalla de participantes permite añadir manualmente o importar miembros del grupo.

## Regla de producto

Los grupos son espacios privados. Las funciones principales viven dentro del grupo:
- Inicio resume lo importante.
- Calendario organiza quedadas y eventos.
- Finanzas resuelve gastos y deudas.
- Torneos organiza competiciones.
- Más queda para miembros, invitaciones y ajustes.
