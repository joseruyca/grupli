# Grupli v15.30.1 — Cohesión visual final de Agenda

Esta versión ajusta la Agenda para que use el mismo lenguaje visual que Inicio.

## Cambios

- La tarjeta de **Próximo plan** en Agenda usa el mismo componente visual que Inicio.
- Se eliminan los círculos decorativos del hero de Agenda.
- Se unifica la jerarquía visual del próximo evento.
- Se añaden métricas separadas en tarjetas mate para no mezclar resumen y plan principal.
- Se suaviza la paleta global con colores más mate:
  - navy/petróleo más sobrio;
  - ámbar menos chillón para Agenda;
  - verde más suave para Finanzas;
  - coral/rojo más mate para Torneos;
  - violeta menos saturado para Más.
- Se mantiene la optimización APK de v15.29.5 y el rediseño de v15.30.

## SQL

No requiere SQL nuevo.

## Pruebas recomendadas

1. Entrar a Inicio y revisar la tarjeta de Próximo plan.
2. Entrar a Agenda y comprobar que el Próximo plan se ve igual.
3. Cambiar entre Semana/Mes.
4. Crear un evento desde Agenda.
5. Confirmar Voy/Duda/No desde la tarjeta.
